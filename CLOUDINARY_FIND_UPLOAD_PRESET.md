# 🔍 How to Find Upload Preset Settings in Cloudinary

## Updated Guide (2025)

Cloudinary's interface may have changed. Here are **MULTIPLE ways** to find upload presets:

---

## Method 1: Direct Link (EASIEST!) ⭐

### Just click this link:

```
https://console.cloudinary.com/pm/settings-tab/add-upload-preset
```

This will take you DIRECTLY to the "Add Upload Preset" page!

**No searching needed!** Just click and fill in the form.

---

## Method 2: Through Settings Menu

### Step-by-Step:

1. **Login to Cloudinary Dashboard**
   ```
   https://console.cloudinary.com/console
   ```

2. **Look for Settings icon**
   - Top-right corner of the page
   - Looks like: ⚙️ (gear icon)
   - OR might be labeled "Settings"

3. **Click on "Settings"**

4. **You'll see tabs:**
   ```
   ┌─────────────────────────────────────────┐
│  ☁️ Cloudinary Settings                │
   │                                         │
   │  [Account] [Upload] [Security] [More] │
   │   ↑                                     │
   │   Click "Upload" tab!                  │
   │                                         │
   └─────────────────────────────────────────┘
   ```

5. **Click "Upload" tab**

6. **Scroll down** to "Upload presets" section

7. **Click "Add upload preset"** button

---

## Method 3: Through Media Library

### Alternative Path:

1. **Go to Media Library**
   - Click "Media Library" in left sidebar
   - Or go to: https://console.cloudinary.com/console/c:c2e6d24f54125a3698c38f78143181/media_library

2. **Look for Settings/Upload Settings**
   - Might be in top menu
   - Or in dropdown menu

3. **Navigate to Upload Presets**

---

## Method 4: Search in Dashboard

### Use the search feature:

1. **In Cloudinary Dashboard**
2. **Look for search bar** (top of page)
3. **Type:** "upload preset"
4. **Click on the result**

---

---

## 🎯 Fill In These Values:

### Required Fields:

**Upload preset name:**
```
sfactor_gallery
```
- Exactly this spelling
- No spaces
- Lowercase with underscore

**Mode:**
```
Unsigned
```
- NOT "Signed"
- MUST be "Unsigned"
- This allows uploads without authentication

**Folder (optional but recommended):**
```
sfactor
```
- Organizes your files
- Creates a folder in Cloudinary
